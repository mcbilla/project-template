protostuff是一个开箱即用的序列化库，使用非常简单，相对其它序列化库，protostuff优势在于：
1. 使用简单。Protobuf每次要编写接口定义文件，然后还要编译，操作太繁琐；
2. 高性能。相对json等文本序列化库，protostuff是二进制的，因此性能比json等方式高；

可以说protostuff在高性能和使用成本上做了很好的取舍。
